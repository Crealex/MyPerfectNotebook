# My Perfect Notebook

Application de prise de notes développée en React avec TypeScript.

## Stack technique

- React 19 + TypeScript
- Vite + SWC
- Tailwind CSS v4
- shadcn/ui
- Supabase (à venir)

## Développement

```bash
bun install
bun run dev
```

## Phases du projet

1. MVP local (localStorage)
2. Backend + DB (Supabase)
3. Features avancées
4. Déploiement production
